{# Accounts#}
If you are logged in then you can view your username at the top of the page to the left of the logout button: it will say Welcome [Username]. While this is not an action a user can make within the site you can contact support: thoughtbasehelp@gmail.com where there will be a 30 day cool-off period (in case you change your mind) before all your personal data will be deleted.
{# Boards#}

Boards

From the home page you click the blue ' Board' button in the top left corner. You can always access the home page by clicking the 'Thought Base' title at the top of the page. Alternatively you can make a new board via this link.
From the home page (accessed by clicking the 'Thought Base' page title) you can click the notepad icon there is a icon in the board section of the tag bar which you can click and then confirm by ticking the checkbox. (this cannot be undone)
By clicking on the notepad you can view the board and its ideas, however, you cannot change the name, project size or project date of a board once it has been created. However, you can always change the status of your board from private to public read-only using the toggle switch in the tag bar.
After creating a board there is a toggle switch in the tag bar where you can switch the board between being private, only you logged in can see it, and being public readonly where anyone with the link can view a readonly version of your board. After pressing the switch please wait a couple seconds for it to take effect.
To print the ideas press the icon in the tag bar to open a new page which will layout the ideas and their data in a more printer friendly way, on this page just use your browser to print the page or press control/command + P.
{# Ideas#}

Ideas

Once you are viewing a board [see: How do I edit the details of a board?] you can see all of its ideas if it has any. To add a new one there is a button in the top left corner Idea to create a new idea. The idea must then have a name (30 characters or less). It can also have a description of up to 1000 characters, a colour and be added to any of the board's tags.
Much like a board you use the notepad icon where a page containing all the data already stored about the idea will appear. You can change any of the data as long as the idea still has a name but if you do not press the blue Save button at the bottom then your changes will be discarded.
There are two ways to do this. 1: when viewing a board of ideas you can press the icon in the top right of an idea or you can press the notepad icon to the left of it and then press the Delete button. A page with a checkbox will open which you must tick to confirm you can to delete the idea (This cannot be undone)
{# Tags + Suggestions#}

Tags & Suggestions

Tag controls are separated from tags in an aquamarine box here the button should be pressed to make a new tag. A tag needs a name and will automatically be given a colour, however, this can be changed to suit the user's preference.
At this point in time the name or colour of a tag cannot be changed once it has been created, the only solution to this is to delete it and create a new one and re-assign any ideas. Once a tag has been created it can be added to ideas, to do this begin editing an ideas [see: How do I edit an idea?] and the user can select tags using the multiple select box before saving the idea again. Once ideas have been added to tag then the board can be filtered using the tag buttons to only show ideas containing that tag. The 'All' button shows all ideas and the 'Colours' button shows all ideas with a colour not set to black.
By clicking on the icon in the tag bar in the lightblue/green box you can then select which tag you want to delete and tick the confirmation box to delete it. All ideas with that tag will no longer be linked to it.
If you have some ideas with colours then the algoithm will generate related colours for you, to view these press the icon on the tag bar where the first 1000 suggestions will be displayed. You can click on any colour to start making a new idea with that colour.
