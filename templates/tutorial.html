Logging in & Registering


Registering for an account

There is a register button on the login page where you can create an account. The details you need to put in are:

  • Username: must only have letters, numbers or underscores and be 20 characters or less
  • Email: must be valid and you should have access to it in case you need to reset your password
  • Password: must be 8-20 characters, contain a lowercase, uppercase, number and special character and match the confirm password
  • User type: select a use case which best describes your use for the site, it will help direct future development plans

Login

You can then login with your username and password. If you forget either of these there are links on the login page to reset your password or get a username reminder via email.

Boards


A board is where you store your ideas for a particular project or event

A board is created by clicking the + Board button in the top left corner of the board page and it needs 3 pieces of data:

  • Name: must be 30 characters or less
  • Project size: select small, medium or large to understand the kinds of data being stored for future development
  • Project date: a date which orders how boards are displayed in your account from ones earliest to latest in date

You can also delete a board by pressing the icon to edit it:

And then the delete board button in the top left corner

When you have clicked the icon to edit the board you can create new ideas and delete them using the + Idea in the top left and the trashcan on each idea

Fianlly, you can always click on the Thought Base title at the top of the page to return to the home page with all your boards.

Ideas


An idea is a record of data which you make within a board. Each one has several points of data which can be recorded:

  • Name: must be 30 characters or less
  • Content: this is an optional description where you can store upto 1000 characters about the idea
  • Colour: this is a colour picker which may appear different depending on your browser, the default is black which will not be recorded. This data feeds into the colour suggestion feature of the site.
  • Tags: this is a multiple select field where you can add and remove user defined tags to/from each idea, none will appear if you have not already created any.

Tags


Tags are a structure to organise your ideas and filter them. The bar at the top or side of your screen contains the current tags and when clicked only ideas with that tag are displayed.

They can be created and deleted using the + and trash icons

The flask opens the colour suggestions page


Colour Suggestions


This page shows all the related colours to the colours recorded in ideas on the current board, excluding black

You can click on any colour to start making a new idea with that colour
